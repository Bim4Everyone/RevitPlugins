# RevitSetCoordParams (Заполнить параметры СМР)
Плагин автоматического заполнения параметров, которых выбирает пользователь,
для всех геометрических элементов модели, исходя из заранее созданных объемных элементов координационного или локального файла

# Сборка проекта
```
nuke compile --profile RevitSetCoordParams
```

# Публикация проекта
```
nuke publish --profile RevitSetCoordParams
```
