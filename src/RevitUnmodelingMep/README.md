# RevitUnmodelingMep (Настройки немоделируемых)
Позволяет настраивать расчетные формулы для элементов которые не моделируются, но присутствуют в спецификации, а так же выполняет их генерацию в отдельный рабочий набор. 

# Сборка проекта
```
nuke compile --profile RevitUnmodelingMep
```

# Публикация проекта
```
nuke publish --profile RevitUnmodelingMep
```
