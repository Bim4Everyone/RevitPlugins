# RevitPylonLoadAreas (Грузовые площади)

Плагин содержит две команды:

1. **Грузовые площади** (`FindPylonLoadAreasCommand`) — строит диаграмму Вороного по выбранным пилонам и стенам (стены разбиваются на точки с заданным шагом), обрезает ячейки по контуру выбранной плиты с учётом отверстий, рисует FilledRegion на активном виде и записывает площадь каждого пилона (в м²) в системный параметр «Комментарии».
2. **Импорт LandXML** (`ImportLandXmlCommand`) — пока не реализована.

# Сборка проекта
```
nuke compile --profile RevitPylonLoadAreas
```

# Публикация проекта
```
nuke publish --profile RevitPylonLoadAreas
```
