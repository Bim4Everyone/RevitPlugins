# RevitMarkAllDocuments (Маркировать всё)
Плагин для сквозной маркировки элементов одной категории в нескольких файлах.
Плагин создаёт json с сквозными значениями марок элементов по нескольким файлам Revit 
и заполняет эти значения у параметров в открытом файле.

# Сборка проекта
```
nuke compile --profile RevitMarkAllDocuments
```

# Публикация проекта
```
nuke publish --profile RevitMarkAllDocuments
```
