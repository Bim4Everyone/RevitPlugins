# RevitDocumenter (Доп.Арм. Размеры)
Плагин предназначен для создания размеров на виде в плане между зонами дополнительного армирования и осями.
Зона дополнительного армирования должна быть выполнена категорией Несущая арматура, 
содержать опорные плоскости с определенными именами, тип связи плоскостей - Слабая.
После запуска пользователь может выбрать в GUI характеристики зон доп армирования и типоразмер размера, который будет использован.
Плагин анализирует взаимные положения именованных опорных плоскостей зоны и сонаправленных осей, определяет ближайшие пары для каждого направления
и строит размеры. 
В случае выбора функции точного построения будет произведен экспорт изображения в png, анализ свободных пикселей и перемещение 
размеров после расстановки в более корректные положения.

# Сборка проекта
```
nuke compile --profile RevitDocumenter
```

# Публикация проекта
```
nuke publish --profile RevitDocumenter
```
