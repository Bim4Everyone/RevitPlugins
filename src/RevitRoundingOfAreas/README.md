# RevitRoundingOfAreas (Рассчитать площади (соц. объекты))
Плагин для автоматического округления системной площади в текстовый параметр. Для социальных объектов 

# Сборка проекта
```
nuke compile --profile RevitRoundingOfAreas
```

# Публикация проекта
```
nuke publish --profile RevitRoundingOfAreas
```
