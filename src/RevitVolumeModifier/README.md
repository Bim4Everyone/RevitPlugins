# RevitVolumeModifier (Модифицировать объемный элемент)
Плагин модифицирует объемные элементы Revit - объединяет несколько в один, разделяет один на несколько, вычитает один из другого

# Сборка проекта
```
nuke compile --profile RevitVolumeModifier
```

# Публикация проекта
```
nuke publish --profile RevitVolumeModifier
```
