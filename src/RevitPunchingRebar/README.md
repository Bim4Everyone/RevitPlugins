# RevitPunchingRebar (Поперечное армирование)
Плагин предназначен для автоматизированной расстановки элементов поперечного армирования в фундаментных плитах и плитах перекрытия

# Сборка проекта
```
nuke compile --profile RevitPunchingRebar
```

# Публикация проекта
```
nuke publish --profile RevitPunchingRebar
```
